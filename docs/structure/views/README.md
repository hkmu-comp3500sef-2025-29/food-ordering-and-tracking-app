# Views module (`src/views`)

This module contains EJS templates for rendering HTML views.

## `/admin`

This directory contains EJS templates for rendering admin dashboard views.

## `/components`

This directory contains EJS templates for rendering HTML components, such as navigation bar and sidebar.

## `cart.ejs`

This EJS template renders the cart page, including the cart items and checkout form.

## `customize.ejs`

This EJS template renders the customize page, including the dish details and customization options.

## `home.ejs`

This EJS template renders the home page, including the start button to create a new session and enter the menu.

## `menu.ejs`

This EJS template renders the menu page, including the list of available dishes.

## `record.ejs`

This EJS template renders the record page, including the list of past orders.
