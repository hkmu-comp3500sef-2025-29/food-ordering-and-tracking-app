# Client Scripts (`src/scripts`)

This module contains client-side scripts for the application.

## `/cart`

This directory contains client-side scripts for the cart page, including the logic to render part of the UI layout and the client interaction with the server.

## `/customize`

This directory contains client-side scripts for the customize page, including the logic to add item into the cart.

## `/home`

This directory contains client-side scripts for the home page, including the logic to create a new session for the user and entering the menu.

## `/record`

This directory contains client-side scripts for the record page, including the logic to render part of the UI layout and the client interaction with the server.
